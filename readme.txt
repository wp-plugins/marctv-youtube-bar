== Description ==

Displays a list of videos from a youtube account. 

== Installation ==

* Install plugin
* Activate it
* edit jquery.marctv-youtube-bar-init.js according to your needs.
* optional: install marctv-jquery-colorbox plugin to open videos in a lightbox
